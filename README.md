## Objetivo do Desafio
O desafio propõe a criação de um **caderno temático no NotebookLM** sobre um **assunto financeiro introdutório**, utilizando a Inteligência Artificial como ferramenta de aprendizagem ativa.
O foco principal é desenvolver habilidades de:
- Curadoria de fontes confiáveis
- Pensamento crítico
- Organização do conhecimento
- Exploração estratégica de prompts com IA
